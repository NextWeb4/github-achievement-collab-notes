# GitHub Achievement Collaboration Notes

アカウント間で実際に行った、レビュー可能な GitHub コラボレーションを記録するための小規模なドキュメントリポジトリです。

## 目的

Pull Request の作成、文書変更のレビュー、マージ、最終コミットの確認など、公開履歴で検証できる共同作業を短いノートとして残します。文書だけで構成されるため、各変更をすばやく確認できます。

## コラボレーション規則

- 公開 README の変更には Pull Request を使用します。
- 例は GitHub 上で実際に行われた操作に限定します。
- 1 つの Pull Request には、独立してレビューできる文書変更を 1 件だけ含めることを優先します。
- 空コミットや誤解を招く活動を作らず、マージ結果を記録します。
- バッジや貢献イベントは、GitHub Achievement の付与を保証しません。
