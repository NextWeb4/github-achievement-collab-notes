## 项目用途

本仓库保存可在公开历史中核验的协作笔记，包括创建 Pull Request、审阅文档改动、完成合并以及检查最终提交。仓库只包含文档，因此每次变更都能快速检查。

## 协作规则

- 公开 README 改动使用 Pull Request。
- 示例必须对应 GitHub 上真实发生过的操作。
- 每个 Pull Request 尽量只包含一个可独立审阅的文档改动。
- 记录合并证据，但不制造空提交或误导性的活跃记录。
- 徽章或贡献事件不等于 GitHub 一定会授予某项 Achievement。
